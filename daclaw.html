Daclaw - Custom macropad keyboard

Nov 26, 2023

  • Custom overall design, 3D printed (I needed 7 iterations).
  • Kaihl choc switches, low profile keycaps
  • Cheap Pro Micro
  • Hand wired (no PCB). This was one of the main motivations to try this.
  • No QMK, just plain Arduino code using keyboard.h library

STL files | Arduino code

Future improvements:

  • Needs some metal plate inside to give some weight.
  • The keyboard emits keys F13 to F24 (the bottom key acts as a modifier). I only configured some programs to use it, needs more software configuration.
