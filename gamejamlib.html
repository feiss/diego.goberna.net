Pixel Sandbox (2D Shadertoy)

Dec 2, 2023

For the GMTK Game Jam 2023 I remade the first level of Mario Bros with a twist. The theme was "Roles Reversed", and the idea was to make the player help the enemies (by feeding them), not kill them.

You can play the cursed game at itch.io.

My game got desqualified for copyright infridgement (surprise!). The fact is that I always thought of repainting all the sprites before submitting, but at some point getting close to the deadline I had to choose between delivering a playable game with full gameplay that would probably be desqualified or a broken game with original sprites.. and I chose the former ¯\_(ツ)_/¯.

One very interesting thing about this mini project was to create the tiniest library I could use for mini game jams like this. Very lightweight, simple, for pixelart (palette-based) style.

You can take a look at the library in the src folder of the GMTK2023 repository.

Features:
  • Pixel art style with palette indexing
  • Basic primitive drawing: pixels, lines, circles, texts, rectangles and images
  • Easy keyboard and mouse handling
  • Multiple animations per spritesheet (frames, fps, custom anchor position)
  • Math utilities: Vector, Rect
